WP Quote Of The Day WordPress Plugin lets you add a welcome overlay containing a daily quote and banner ad to your WordPress Website (Just like Forbes!)

== Description ==
This plugin lets you add a Quote of the Day widget to your WordPress page. Select your favorite topic Such as Business Quotes, Success Quotes, Power Quotes, Food Quotes, Art Quote, Funny Quote, Love Quote or Inspirational Quotes. The plugin will automatically display a new famous & spiritually moving quote on your welcome screen, each and every day!

Want to monetize the overlay? No Problem! You can add your choice of banner advert or JavaScript/iFrame/HTML ad network ad tag (AdSense supported) that will appear just below the quote. Sell this premium ad spot to your sponsors and make money in an elegant and professional way.

[WP Quote Of The Day](http://wpquoteoftheday.com) is the web's largest quotation api website containing over 550,000 quotes from 125,000 authors orginized in 125 topics. With this huge collection of quotes you can expect a fresh Quote of the Day that educates, entertains and wow's your users.

= Features =
* Plugin will create an overlay with a daily quote and banner ad spot that will welcome your users
* Choose the category which best matches your website (out of 125 quote topics)
* Monetize with your own banner ad or add an html/javascript/iframe ad tag (such as AdSense)

= Pro (Paid) Features =
* Fully customize the look of the overlay (background color, opacity)
* Choose which Google font you would like to have for the quotes and authors
* Specify font size and colors
* Specify the countdown duration
* Specify the countdown text
* Choose if you would like to automatically redirect users after a certian amount of seconds
* Choose the destination URL of the countdown text link
* Request a new quote
* Enable/Disable if to show the circle around the quote and spcify the circle color
* Set frequency capping (how often the end user should see the welcome screen)
* Set which types of pages should the plugin create an overlay for

== Installation ==
1. Upload `wp-quote-of-the-day.php` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` menu in WordPress.
1. Go to 'WP Quote Of The Day' Plugin admin settings (in the admin sidebar)
1. Select a category (quote topic) that best matches your website
1. Optional: go to banner setting to add your banner ad or AdSense ad tag.

== Frequently Asked Questions ==
= How many quotes are in your database? =
We have over 550,000+ quotes in our database from over 125,000 authors, categorized in 125 categories.

= How are the quotes chosen? =
Once you specify the quotes category you would like to subscribe to we will randomize the quotes and display a new quote on your site each day. If you do not like the quote you have an option to request a new quote from our database using the plugin admin panel.

= What type of banner can I insert in the overlay? =
You can choose to use either a static image banner with a destination URL or a rotating adtag such as AdSense, Media.net or any other ad network tag. (iFrame / JavaScript)

= Can this hurt my SEO efforts? =
There is no evidence that this plugin or any similar plugin have negative impact on your SEO. In fact – the plugin and this website have been proven (tested) to be fully compliant with Google’s mobile site compliance requirements.

= What versions of WordPress does this work with? =
We tested with 4.7.5, 4.6.1, 4.4.2, 3.8.1, 3.7.1, and 3.6.1 but it probably works with other versions (3.x or greater).  

Let us know in the "Compatibility" poll if it does work with a particular version, and let us know (via the Support tab) if there are any other issues. Thanks!
